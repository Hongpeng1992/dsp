Convert complex FFT data to a magnitude only spectrum in-place.

Parameters
[in]cplxFft- complex data vector containing normalised complex FFT.
[in]zeroUnused- zero unused BINs in complex vector.

After calling this only the real part of each complex value is valid, the imaginary part will be 0 because the power is spread over a positive and negative half spectrum we multiply the value in each BIN by 2 to get the correct amplitude. The negative half of the FFT is not for use so only the first N/2 values represent the real FFT spectrum.

inlinestatic

Convert complex FFT data to power spectrum in-place.

Parameters
[in]cplxFft- complex data vector containing normalised complex FFT.
[in]zeroUnused- zero unused BINs in complex vector.

The power spectrum is computed as re[n]^2 + im[n]^2. Only the real part of each complex value is valid, the imaginary part will be 0. The negative half of the FFT is discarded so only the first N/2 values represent the real FFT spectrum.
